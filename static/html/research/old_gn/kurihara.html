プロキシサーバによる効率的なWeb閲覧履歴の取得に関する研究


利用者が計算機を用いて作業を行う場合，そこには利用者固有の利用傾向が存在します．計算機が利用者の利用傾向を把握することで，閲覧頻度の高いWebページをあらかじめ取得しておくことや，以前のデスクトップの状態を復元したりといった，利用者にとって利便性の高いサービスを提供できます． そこで，利用者のWeb 閲覧履歴を把握するために，Private Server という，利用者計算機上で動作する個人専用のプロキシサーバを開発しました．Private Server は外部計算機との通信を仲介し，通信履歴を保存します．Private Server を用いることにより，利用者計算機上の情報を外部に漏らすことなく，計算機外部との通信履歴を扱えます． しかし，Private Server により保存された履歴情報から利用者のWeb 閲覧履歴を把握するためには，不要な情報が多く，必要な情報を抽出する方法が必要になります．

【特徴】

  1. ブラウザに依存しない履歴収集が可能
  2. 高い再現率と適合率を実現
  3. 個人に合わせた形態で収集可能
