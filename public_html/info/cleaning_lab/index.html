研究室の掃除について¶

乃村研では，研究室のメンバで106号室の掃除を行っている．
ここでは，研究室の掃除として，ゴミ捨て，掃除機がけ，大掃除について述べる．

ゴミ捨て¶

ゴミ捨ては毎週月曜日と木曜日の午前10時から11時の間に行う．
なお，集積所に持っていくゴミは月曜日と木曜日で異なっている．
このため，一覧を書いた紙を確認して持っていくこと．

掃除機がけ¶

掃除機がけは毎月最初の週の月曜日に行う．

大掃除¶

大掃除は年度末に行う．
対象となる部屋は106号室と105号室の流し周辺である．
  1. 当日までにやること
    1. 各自の持ち物
      各自のいらないものは，持って帰るか処分すること．
      また，研究室においておくものは，箱などにまとめて入れておくこと．
    2. 机とイス
      机，イスが足りるか否か確認する．
      また，来年度の机とイスの配置について相談する．
    3. PC
      PCを引き継ぐ人は，引き継げる状態にしておく
    4. 確認事項について
      1. 掃除を行う日時を連絡する．
      2. 当日のPCの置き場所の確認をする．
      3. 廊下に物を置いて良いか確認する．
      4. 当日参加する人の確認をする．
  2. 当日に行うこと
    1. 処分する物について
      処分する物は廊下に出す．
      この際，乃村先生に廊下に出して良いか確認する．
    2. PCについて
      PCは105号室に移動する．
    3. 106号室の掃除手順について
      1. 窓の掃除を行う．
      2. 机を部屋の北側に移動し，掃除機をかけた後に雑巾がけをする．
      3. 机を部屋の南側に移動し，掃除機をかけた後に雑巾がけをする．
      4. 机の配置を整えた後に各自PCのセッティングを行う．
    4. 105号室の掃除手順について
      流し周辺の掃除を行う．
