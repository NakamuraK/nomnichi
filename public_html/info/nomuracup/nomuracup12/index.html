ルール

  1. 1チーム4～3人のチーム戦で行う．チームは，当日決定する．
  2. テーマに沿った写真をチームごとに撮影する．
  3. テーマは，5つ用意し，開始前に発表する．
  4. 各テーマ毎に各チーム1枚ずつベストショットを選び，5枚の写真のデータをUSBにいれて牛尾に提出する．
  5. チーム全員で行動し，移動手段は徒歩限定とする．
  6. 各チームは，最低30分に一度は近況をツイートする．
  7. 制限時間内に405号室に戻って来なかったチームは，ペナルティとして1ポイント減点する．

写真の評価方法

  1. 撮ってきた写真は，ゲスト審査員5人により審査してもらう．
  2. 各テーマ毎区切りに審査を行う．
  3. 写真を１枚ずつプロジェクタを表示し，各チームの代表者は，その写真について最大30秒のプレゼンテーションを行う．
    4. 代表者は，テーマ毎に交代し，チーム全員が一度以上は，プレゼンテーションを行う．
  4. プレゼンテーションの順番は，じゃんけんで決め，その後ローテーションする．
    5. (例) テーマ１は，B->C->Aの順番でプレゼンテーションする．テーマ２は，C->A->Bとなる．
  5. 各チームのプレゼンテーション終了後，審査員は，3枚の中から一番テーマに沿っていると思う写真を選ぶ．選ばれたチームは，審査員ひとりにつき1pt獲得する．
  6. 5テーマ合計で一番ポイントを集めたチームの優勝とする．
  7. ポイントを集計した結果，複数のチームが同点となった場合は，各チーム新たに写真を1枚選出し，プレゼンテーションと審査を行う．

テーマ

乃村杯という名前にちなんで，以下の5つのテーマとする．
  1. の：乗り物
  2. む：昔の物
  3. ら：楽しい物
  4. は：春の訪れを感じさせる物
  5. い：生き物
