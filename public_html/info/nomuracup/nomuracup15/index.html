持ち点　　　　　　25000点
点数は3万点返しのトータルで計算する．

例
37000点の場合，37000 - 30000 = 7000 → +7
19000点の場合，19000 - 30000 = -11000 → -11

端数は五捨六入する．

ウマ　1位 +10 2位 +5 3位 -5 4位 -10
オカ トップ賞
　例
　　1位：40000点 -> +10 +10(ウマ) = +20
　　2位：31000点 -> +1 +5 (ウマ) = +6
　　3位：19000点 -> -11 -5 (ウマ) = -16
　　4位：10000点 -> -20 -10(ウマ) = -30
この場合，+ の合計が26，- の合計が46であり，
+ と - の差である20がオカ(トップ賞)となるので
1位： +20 +20 = 40 となる．

よって，1位：+40　2位：+6　3位：-16　4位：-30 となる．

この点数の3回の合計で最終的な順位を決定する．

誰か1人が飛んだ場合，その時点での点数で計算し，終了する．
(ハコ下計算 例 -8000 → -38)

東風戦開始後80分経過時点で終わっていない場合，
現在の局終了時点で．点数を計算し終了する．
