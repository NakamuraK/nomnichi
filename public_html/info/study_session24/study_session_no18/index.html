Latex文書作成手順

  1. template.texの内容をLatexファイル(ここでは，fukai01.texとする)にコピー
  2. fukai01.texを開き，以下の作業を行う
    1. 「はじめに」の内容をabstract{}の中に書く．
    2. 資料のタイトルを\title{}の中に書く．
    3. 資料作成者の氏名を\author{}の中に書く．
    4. 資料提出日を\date{}の中に書く．
    5. \startsectionの括弧に左から{開始 の章番号}{開始の節番号}{開始の章題目}を書く．
    6. 章題以下の本文を\startsectionより下に書く
      7. 以下に例を示す． 
            %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
      \title{Haskell勉強会資料}%タイトル
      \author{谷口研究室 深井貴明}%氏名
      \date{2012年6月10日}%日付

      \include{begin}%begin{document}の代わり
      \mymaketitle%タイトル生成
      \abstract{
        ここに「はじめに」の内容を書く(sectionなどは不要)
      }

      \startsection{4}{2}{Syntax in Functions}
      \subsection{Guards, guards!}
      本文を書く．
      \include{end}%end{document}の代わり
      %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

Latex文書の作成時の注意

Latex文書を作成する際の注意点を以下に示す．
  1. \begin{document}, \end{document}を書かずに，\include{begin}, \include{end}を書く．
  2. タイトルは\mymaketitleを用いて作成する．
    {使い方は\maketitleと同じ，template.txtにすでに書いてある．}
  3. \include{begin}より上にtitle，authorおよびdateの設定以外を書かない．
  4. 参考文献を書く際，\end{bibloigraphy}の前後には空白文字を入れない．(空白文字を入れると全体のPDF作成時にエラーとなる．)
  5. 必ずLatex文書をPDFに変換できることを確認する．変換手順については後述する．

ファイルの命名規則

画像ファイルを追加する際には，以下の命名規則に従ってファイル名を決める． 
[自分の名前][資料番号]_[任意の文字列].eps
ただし資料番号はその人が提出した資料の回数とする． 勉強会の回数ではない． 
例:深井が2回目に提出したファイル

fukai02.hogehoge.eps
.bbファイルを追加するときも同様の命名規則に従う．

個人のLatex文書をPDFに変換する手順

個人のLatex文書をPDFに変換するには，以下のコマンドを実行する． 
$ platex fukai02.tex
$ dvipdfmx fukai02.dvi

全体のLatex文書をPDFに変換する手順

すべてのLatex文書をつなげたPDFを作成するには，以下のコマンドを実行する． 
$ platex haskell_workshop.tex
$ dvipdfmx haskell_workshop.dvi

リポジトリへのプッシュについて

リモートリポジトリへプッシュするファイルは，以下の3つである．
  1. 作成したLatexファイル(.tex)
  2. 作成した文書の内容を反映した全体のPDFファイル(.pdf)
  3. Latex文書で用いた画像ファイル，.bbファイル
