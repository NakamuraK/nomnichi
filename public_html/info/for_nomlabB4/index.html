Twitterについて¶

乃村研究室では研究室内の連絡手段としてTwitterを利用している．
  1. Twitterアカウントの作成
    Twitter:https://twitter.com/
    Twitterの研究室用アカウントを作成する．
    アカウント名(例:Yoshii_Hi 吉井英人の場合)
    注意:ツイートを非公開にする．
    設定->ツイートの公開設定で「ツイートを非公開にする」にチェックを入れる．
  2. 乃村研究室メンバーのフォロー
    乃村研究室の現役・OPをフォローする．
    フォロー手順は以下の通りである．
    1. 乃村先生の研究室内アカウント(@nom4476_sec)をフォローする．
    2. 乃村先生のlaboリストのメンバーをフォローする．

プリンタ接続¶

  1. ドライバのダウンロード
    名称:brother hl-5270dn
    URL:http://solutions.brother.co.jp/public/dlf/download_top.html?prod=hl5270dn
    ダウンロード手順
    1. OSの選択: Windows 7
    2. タイプの選択: ドライバー
    3. フルパッケージダウンロード
  2. ドライバのインストール
    インストール手順
    1. ドライバの選択: Brother HL-5270DN series
    2. インストール方法: 標準インストール
    3. 接続方法: Brother ピアツーピア ネットワークプリンタ
    4. プリンタの選択: ネットワークを検索し，リストから選択
      ノード名:BRN_E19F2Dを選択
  3. 設定
    設定手順
    1. コントロールパネル->デバイスとプリンターの表示->Brother HL-5270DNseries->印刷オプション
    2. 仕切り線: なし
    3. 給紙方法: MP トレイ

wmf2epsの設定¶

  1. Win7用のPSプリンタドライバをダウンロード．
    URL:http://www.wolf-s.homepage.t-online.de/wmf2eps/win7.htm
  2. ダウンロードしたWMF2EPS1.32-sをProgram Files以下にコピー
  3. ダウンロードしたWin7をWMF3EPS1.32-s下にコピー
  4. プリンタの追加
    コントロールパネル->デバイスとプリンターの表示
    1. 右クリック->プリンターの追加
    2. インストールするプリンターの種類選択: ローカルプリンターを追加します
    3. プリンターポートの選択: 既存のポートを使用(LPT1:(プリンタポート))
    4. プリンタードライバーのインストール: ディスク使用
    5. フロッピーディスクからインストール: Program Files->WMF2EPS1.32-s->PSprint->Win7->W7Print
    6. プリンタ名: WMF2EPS Color PS L2
    7. プリンター共有: このプリンターを共有しない
    8. 通常使うプリンターに設定するにチェック
  5. プリンタの設定
    1. コントロールパネル->ハードウェアとプリンターの表示
    2. WMF2EPS Color PS L2の印刷設定を開く
    3. 用紙/品質の詳細設定を開く
    4. 以下の内容を変更
      1. TrueTypeフォント: ソフトフォントとしてダウンロード
      2. PostScript出力オプション: EPS
      3. TrueType フォントダウンロードオプション: ビットマップ(アウトラインの方がいい?)
      4. PostScriptエラーハンドラーを送信: いいえ
    5. wmf2epsを起動
      1. EPS-Conversion->Conversion Setupをクリック
      2. EPS-pocessingのPS-Printerを「WMF2EPS Color PS L2」にする．
    6. WMF2EPS SettingsのStandard pathsを変更
      1. Use Standard path for storing EPS-filesにチェックを入れる
      2. EPS-Standard PathとWMF-Standard Pathを任意のものに変更
        例:C:¥User¥Administratior¥Documents¥eps¥
