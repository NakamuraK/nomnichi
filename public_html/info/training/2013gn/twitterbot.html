TwitterBot

TwitterBotプログラムの入手方法

  1. GitHubにアカウントを作成します．
  2. 作成したGitHubアカウントを，nomlabに登録します．
    この登録は，M1あるいはM2に頼む必要があります．
  3. nomlabが持っているリポジトリ GN-B4-Exercises.git を自分のGitHubリポジトリにForkします．
  4. 課題の作成環境で以下のコマンドを実行します．
    ただし，hogehogeは自分のユーザ名に変更してください．
    $ git clone https://github.com/hogehoge/GN-B4-Exercises.git

認証

TwitterBotを動作させるためには，認証が必要です．認証の手順は以下のとおりです．
自分のTwitterアカウントへのログイン後，
http://www.swlab.cs.okayama-u.ac.jp/lab/nom/info/training/2012gn/oauth_script.rb
にあるスクリプトを使用し，次の手順で行います．
  1. 「Access this URL and approve」の後に示されるURLに，ブラウザからアクセスする
  2. 「許可」ボタンを押す
  3. 「Input OAuth Verifier」と表示して入力待ちの状態になっているスクリプトに「許可」ボタンを後に表示される7桁の数字を入力する
    上記の操作終了後，表示されるAccess tokenとAccess token secretと以下の2つを合わせた計4つの文字列を使用します．
    Consumer token : tuwAf85K7bJ6OqWG3Eomg
    Consumer token secret : MokGhmoBR8XTa17wYkNT67rK6gHrUEgFGiPZbbGYT8
  4. 4つの文字列を，setting.yml に書き込みます．
  5. setting.yml と同じディレクトリで以下のコマンドを実行します．
    $ git update-index --assume-unchanged setting.yml
    これは，4つのトークンを誤って外部に公開してしまわないようにするためです．
    以下のコマンドで正しく設定が変更されたか，確認します．
    $ git ls-files -v
    setting.yml に表示される"H"が小文字になっていれば，正しく設定が変更されています．
