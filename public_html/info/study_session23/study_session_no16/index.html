詳細

概要

OpenStackのマニュアルを和訳し，スライドにまとめて発表する．
(マニュアルは2011/5/25時点で最新ものを使用)

使用したマニュアル

OpenStack Compute Administration Manual(PDF版)
OpenStack Object Storage Administration Manual(PDF版)

今回行ったこと

OpenStack Computeのマニュアルの5章から8章までの発表を行った．

感想

＜乃村 能成＞
OpenStack は，いわは総合芸術のようなもので，理解するには幅の広い素養が必要だと一層感じました．勉強しがいのある題材ともいえます．
＜千崎 良太＞
前回の勉強から間が空いていたにもかかわらず，スライドの見直しをしていなかったので，うまく発表できませんでした．次回は，予習して勉強会に参加します．
＜宮崎 清人＞
前提となる知識が十分でなく，自身の発表内容に関して理解できていなかった箇所が多々ありましたが，先生と先輩方が説明をしてくださり，理解することができました．自身の研究では触れる機会のない技術についても学ぶことができ，非常に有意義でした．
＜中原 大貴＞
事前に予習，復習，およびOpenStackのインストールをしておくべきだったと反省しています．次回は自身の担当箇所のスライド修正を含めて準備を怠らないようにします.
＜三原 俊介＞
期間が空いた場合，予習として今までのスライドを見返しておくべきだったと感じました．次回は，スライドと資料の見直しをします．技術に関しては各種技術の復習となる良い題材だと思いました．
＜左海 裕庸＞
OpenStackはOpenStack固有のもの以外の知識も必要となるため，とても勉強になりました．私は前回の勉強会には参加していなかったため，分からない用語がありました．次回は予習して参加します．
＜福田 大志＞
前回から時間が空いていて，忘れてしまっている部分が多くありました．次回はきちんと復習をしておこうと思います．
＜池田 騰＞
OpenStackで使用されている技術に対する知識が不足していることもあり，OpenStackの構成の把握が難しかったです．しかし，先生や先輩方が行ってくださった図解のおかげで全体像を理解をすることが出来ました．次回は予習と復習をして参加します．
＜木村 有祐＞
今回は自分のスライドの発表はなかったですが，他の方のスライド発表を聞き，参考になる事が多いことに気づきました．次回は最初に発表するので，それまでに気づいたことを反映させたスライドを作成しておきます．
＜西田 潤＞
スライドの見直しができておらず，内容だけでなく体裁も直さなければならないところがいくつかありました．指導していただいたところは次回のスライド発表にいかします．
＜吉井 英人＞
分からないところが多かったですが，先生と先輩方が丁寧に解説してくださったので，勉強になりました．全体的に知識が足りないなと感じます．これからどんどん知識を吸収していきたいです．あと，計画性のない行動はだめだと思いました．あとで，後悔します．
