論文執筆の際の注意点

作成者：乃村 能成
作成日：2011/10/20
更新日：2011/10/24 (木村 有祐)

1. はじめに

本文書では，論文を執筆する際の注意点についてまとめている．
以下の規約は情報処理学会用論文に基づくものである．

2. 表記

(1) タイトルは，of や a を除いて，capitalize する

A method for monitoring of PCI devices
→
A Method for Monitoring of PCI Devices

(2) 図表

キャプションは，英語表記の場合 「.」で終わるが，
日本語表記は，何も付けない．

(3) 片仮名

インタフェース
オーバヘッド

(4) 記号の統一

句読点は，「，．」
文中の () は，（）に する
項番の () は半角でもよい．

(5) 送り仮名

(a) 名詞中では省くことが多い

割り込み → 割込み
繰り返し → 繰返し
(例外) 書込み → 書き込み， 読込み → 読み込み

表わす → 表す
取り扱い → 取扱い
受け付けは，→ 受付けは，

(名詞) 切り替え → 切替え
(動詞) 切り替える → 切り替える

対応づ → 対応付


(6) 数詞

ひとつ → 1つ
ふたつ → 2つ
二つ → 2つ
2個所 → 2カ所


(7) ひらがなで表記すべきもの

常に → つねに
上で → うえで
毎 → ごと
例えば → たとえば
従って → したがって
時 → とき
無い → ない
以下の通り → 以下のとおり
但し → ただし
即ち → すなわち
様々 → さまざま
伴う → ともなう
全て → すべて
留め → とどめ
対応を取る → 対応をとる
及び → および

(8) 漢字で書くもの

よい → 良い
など → 等
わかる → 分かる


(9) その他 (言い回しなど)

処理にて → 処理において
半ばにて → 半ばで
(以下，～と呼ぶ．) → (以下，～と呼ぶ) ()内の「．」は取る
～を元に → ～を基に

～したり，～する → ～したり，～したりする
多岐に及ぶ → 多岐にわたる


(10) 引用

句読点は，引用の後に入れる．

～する．[1] → ～する[1]．
