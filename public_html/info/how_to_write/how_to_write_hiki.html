最終更新日：2009年6月2日

はじめに

このページでは，Lastnoteやノムニチで使用されているHikiの記述法を説明する．
基本的な記述法はTextFormattingRulesを見てもらいたい．
また，RedmineのWikiに関してはRedmineにおけるwiki記法を参照してもらいたい．
ほとんどのテキスト入力エリアでtextile記法が使用できる．
以下，ノムニチのみで使える記述法を説明する．

InterWiki記法

bibdata:id             文献DBへの参照
nomnichi:parmalink     ノムニチの他記事へのリンク

plugin記法

{{attachment(name)}}   添付ファイルの参照
{{image(name)}}        添付のイメージ表示
{{asin(number)}}       Amazonへのリンク

その他

//...                  続きを読むのブレーク位置

//<secret
    ～                 ここに書かれた文章を読めるのはログインした人のみ
//secret>
