このページでは，Googleアカウントの使い方や活用法を載せています．

アクセス方法

  1. Googleにアクセスし，右上の「ログイン」をクリックする．
  2. メール，パスワードを入力してログインします．
    • メール: nomura.laboratory
    • パスワード:いつもの
  3. 間違っていなければログインできます．
  4. ログインできない場合は，入力が正しいかどうかチェックしましょう．

Picasaウェブアルバム

  • 使い方

    1. 写真のアップロード
      1. 「アップロード」をクリックし，新たにアルバムを作成するか，既存のアルバムを選択する．
      2. アップロードしたい写真を選択し，アップロードする

  • 活用法

    • 乃村研写真館として利用している．

カレンダ

  • 使い方

    1. 予定の登録方法
      1. 予定を登録したい日時に対応する場所をクリックする．
      2. タイトルを入力し，登録するカレンダを選択する．
      3. より詳細に登録したい場合「予定の詳細を編集」をクリックし，詳細を入力する．
      4. 「予定を作成」をクリックし，予定を登録する．
    2. 予定の削除方法
      1. 削除したい予定をクリックする．
      2. 「削除」をクリックする．

  • 活用法

    • 乃村研でのイベント（ミーティングや乃村杯）を載せ，みんなで行事予定を共有する．
    • 「周期的に発生する作業の発見を支援するカレンダサービス」の研究で使われている．

マップ

  • 使い方

    1. 目印（ピン）の立て方
      1. ピンを立てたい場所を検索する．
      2. 出てきた吹き出しの「マイマップに保存」をクリック
      3. どの地図に保存するか選択する．
      4. タイトル，説明を編集して「完了」をクリックする．
    2. ピンの編集，削除の方法
      1. 編集したいピンが立っている地図を選択
      2. 左下に選択した地図に保存されているピンが表示されるので，編集したいピンを選択する．
      3. 編集，または削除する．

  • 活用法
