TeX環境構築手順¶

このページでは，WindowsとUbuntu 12.10上でTeX環境を構築する手順について示す．(Macは誰か書いて)
なお，乃村研はGitLabを用いて卒業論文，および学会論文の内容を乃村先生にご指導いただく．
このため，文字コードをUTF-8にする必要がある．
このことに注意してTeX環境を構築すること．

Windowsの場合¶

  1. WinShellを取得
    TEXの総合ツール．TeXファイル記述からpdf作成までできます．
    しかし，UTF-8のファイルを変換しようとするとエラーが発生するバグがあるため，3以降の処理を行う必要がある．
    なお，今ではWindowsにも簡単にインストールすることができます．
    http://www.winshell.de/
    "http://www.ms.u-tokyo.ac.jp/~abenori/mycreate/":Windows版インストーラー
  2. 文字コードの変換
    WinShellの「オプション-->フォント」でエンコーディングをshiftJISからUTF-8に変更する．
  3. tex --> dvi
    1. ユーザ指定プログラムの設定
      WinShellの「オプション-->ユーザ指定プログラム」で，
      Tool 1～10のどこかを「主なTexプログラムの設定」の「LaTeX」と同様に設定した上で，以下の点を変更する． | プログラム名 | 自身がわかりやすい名前にする | | コマンドライン | -kanji=utf8 を追加する |
    2. ショートカットをツールバーへ追加
      1. 「オプション-->表示-->ユーザ設定」のカテゴリで「ユーザ指定プログラム」を選択
      2. ショートカットを追加したいユーザ指定プログラムをツールバーにD&D
      3. ツールバーに表示するアイコンを選択，あるいは編集する
  4. dvi --> pdf
    1. ユーザ指定プログラムの設定
      WinShellの「オプション-->ユーザ指定プログラム」で，
      Tool 1～10のどこかを「主なTexプログラムの設定」の「PDFLaTeX」と同様に設定した上で，以下の点を変更する． | exeファイル名 | C:\tex\bin\dvipdfmx.exe(各自の環境による) | | プログラム名 | 自身がわかりやすい名前にする | | コマンドライン | -f msembed.map "%s.dvi" |
    2. ショートカットをツールバーへ追加
      1. 「オプション-->表示-->ユーザ設定」のカテゴリで「ユーザ指定プログラム」を選択
      2. ショートカットを追加したいユーザ指定プログラムをツールバーにD&D
      3. ツールバーに表示するアイコンを選択，あるいは編集する

Ubuntu 12.10の場合¶

  1. パッケージの取得
    日本語LaTeXに対応したtexlive2012と日本語のdviファイルを閲覧できるxdvikを取得する．
    $ sudo apt-get install texlive-lang-cjk xdvik-ja

    なお，Ubuntuのバージョンが低いとtexlive2012がUTF-8に対応していないため，注意すること．
  2. ファイルの編集
    VimやEmacsを使用してTeXファイルを作成する．
  3. tex->dvi
    pLaTeXを実行してTeXファイルをDVIファイルに変換
    $ platex <ファイル名>.tex
  4. dvi->pdf
    dvipdfmxを実行してDVIファイルをPDFファイルに変換
    $ dvipdfmx <ファイル名>.dvi
  5. Makefile
    また，その他諸々の環境が整っている場合，PDFに図を入れるカッコイイ方法にあるMakefileでPDFファイルを作成することができる．
