RVM・Rubyインストール(Debian)¶

RVMインストール(Debian)¶

DebianにRVMのインストール方法は複数ある．
ここでは，そのうちの1つについて述べる．

  1. 以下のコマンドで，rvmをインストールする．
    $ \curl -#L https://get.rvm.io | bash -s stable --autolibs=2
  2. 以下のコマンドでbashを再読み込みする.
    $ source .bashrc

    もしくはDebianを再起動する.
  3. 以下のコマンドで，RVMが正常にインストールされた事を確認する．
    $ rvm -v

    確認できない場合は，PATHを確認する．

Rubyインストール¶

RVM上にRubyをインストールする方法について述べる．

  1. 以下のコマンドで，Rubyをインストールする．以下はバージョン1.9.3をインストールする例である．
    $ rvm install 1.9.3
