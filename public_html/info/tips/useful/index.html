PuTTY¶

PuTTY（パティ）は、Simon Tathamが開発・公開しているターミナルエミュレータです．
"http://yebisuya.dip.jp/Software/PuTTY/":PuTTYごった煮版 などの様々な機能が追加されたものもあります．
http://www.chiark.greenend.org.uk/~sgtatham/putty/

ThunderBird¶

Thunderbird（モジラ・サンダーバード）は Mozilla を起源とし、オープンソースで開発が行われている電子メールクライアントです．
Gecko エンジンを使用し、優れた描画性能、高速性、拡張機能など拡張性に富んでいます．
メールチェッカーと併用するとより便利に使用できるでしょう．
http://mozilla.jp/thunderbird/

Mozilla FireFox¶

Mozilla Firefox（モジラ ファイアフォックス）とは、Mozilla Foundationが開発するオープンソース・クロスプラットフォームのウェブブラウザです．
近年普及率が大幅に増加しています．試しに使ってみてはどうでしょうか．
http://mozilla.jp/firefox/ なお，GoogleChromeもオススメです．

WinShell¶

TEXの総合ツール．texファイル記述からpdf作成までできます．今ではWINDOWSにも簡単にインストールすることができるのでおすすめです．
http://www.winshell.de/
"http://www.ms.u-tokyo.ac.jp/~abenori/mycreate/":Windows版インストーラー

Meadow2¶

Windows版のEmacs．操作性もLinuxのEmacsと似ています．Emacs使いには使えるツールになりそうです．このページもこのツールで書いてます．
http://www.meadowy.org/meadow/

Emacs for Windows¶

上記のMeadowは開発終了していて，emacs21くらいで止まっています．
実はEmacsは，GNUでWindows用のバイナリが配布されているので，「最新のがいい！」という方なんかはこちらを使うといいかもしれません．
http://ftp.gnu.org/gnu/emacs/windows/

TEX¶

レポートや論文を記述するためのツールです．記録書から卒論，学会で発表する論文にまで幅広く使えます．おすすめです．
http://oku.edu.mie-u.ac.jp/~okumura/texwiki/
