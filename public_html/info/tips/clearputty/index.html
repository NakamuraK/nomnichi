透けるPuTTY¶

PuTTYのウィンドウを透過させて，カンニングしながら作業できるようにするには．

1. PuTTY のダウンロード¶

ここ からダウンロードする．
2013/02/28 時点での最新版は，putty-gdi-20120211.zip

2. 日本語化¶

  1. ダウンロードしたPuTTYを展開する．
  2. ja_JP/putty.lng を putty.exe と同じ階層にコピーする．

3. 設定¶

透明度の設定は Window -> Transparence で行う．
ウィンドウのアクティブ時/非アクティブ時の背景の透明度を % 単位で指定してください。
0% が完全な透明で，100% が不透明です。
ただし，Effect: Plane 時には，0% を指定しても僅かに不透明が混じります．
