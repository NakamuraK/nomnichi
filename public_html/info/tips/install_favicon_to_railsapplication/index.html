1. faviconの作成

新しく作成する，あるいは既存の画像から作成するという方法があります．
以下のサイトが使いやすかったです．
X-Icon Editor
16x16, 24x24, 32x32, および64x64の4種類のfaviconを生成することができます．

以下のサイトはその他のfavicon作成用サイトについてまとめてあります．
参考サイト - http://www.ideaxidea.com/archives/2008/02/faviconico.html

2. Railsアプリにfaviconを設置

参考サイト - http://d.hatena.ne.jp/hrendoh/20111102/1320236055

faviconファイルを設置

faviconファイル(favicon.ico)を設置する．
Rails3.1系以上では，app/assets/imagesに設置する．
Rails3.0系では，public/images以下に設置する．

リンクの追加

app/views/layouts/application.html.erbのheadに以下の行を追加する．

  favicon_link_tag 'favicon.ico'
