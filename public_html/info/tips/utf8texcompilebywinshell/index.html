WinShellでUTF-8なtexを文字化けさせずにpdfに変換する方法

WinShell のインストール

だれかおねがい．

1. tex --> dvi

1.1. ユーザ指定プログラムの設定

WinShellの「オプション-->ユーザ指定プログラム」で，
Tool 1～10のどこかを「主なTexプログラムの設定」の「LaTeX」と同様に設定した上で，以下の点を変更する．

プログラム名 自身がわかりやすい名前にする
コマンドライン -kanji=utf8 を追加する

1.2. ショートカットをツールバーへ追加

  1. 「オプション-->表示-->ユーザ設定」のカテゴリで「ユーザ指定プログラム」を選択
  2. ショートカットを追加したいユーザ指定プログラムをツールバーにD&D
  3. ツールバーに表示するアイコンを選択，あるいは編集する
  4. おしまい

2. dvi --> pdf

2.1. ユーザ指定プログラムの設定

WinShellの「オプション-->ユーザ指定プログラム」で，
Tool 1～10のどこかを「主なTexプログラムの設定」の「PDFLaTeX」と同様に設定した上で，以下の点を変更する．

exeファイル名 C:\tex\bin\dvipdfmx.exe(各自の環境による)
プログラム名 自身がわかりやすい名前にする
コマンドライン -f msembed.map "%s.dvi"

2.2. ショートカットをツールバーへ追加

  1. 「オプション-->表示-->ユーザ設定」のカテゴリで「ユーザ指定プログラム」を選択
  2. ショートカットを追加したいユーザ指定プログラムをツールバーにD&D
  3. ツールバーに表示するアイコンを選択，あるいは編集する
  4. おしまい
