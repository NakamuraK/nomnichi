設定手順¶

  1. はじめに
    本手順は，Windowsで設定するときの手順である．
    Macでも同様の方法で設定できる(はず)
  2. 英辞郎の購入
    書店もしくは，Amazonで書籍を購入
    Amazon 英辞郎 第7版
    ここでは，「英辞郎 第6版」を購入
  3. PDIC (英辞郎)のインストール
    英辞郎の付属CDを使って，PDIC (英辞郎)をインストールする．
  4. 辞書データの変換
    • PDIC (英辞郎)を起動する．
    • ツールバーから File -> 辞書設定<詳細>
    • EIJI128 を右クリック -> 辞書の変換 -> PDIC1行テキスト形式に設定
  5. Mouseover Dictionaryのインストール
    http://maru.bonyari.jp/mouseoverdictionary/
    上記サイトからfirefoxのアドオンの「Mouseover Dictionary」をインストールする．
    ここでは，mouseoverdictionary-0.6.14.xpiを選択した．
  6. Mouseover Dictionaryの設定
    • firefoxのツール->アドオン
    • Mouseover Dictionaryの設定で辞書のロードを選択し，PDIC1行テキスト形式しておいた辞書を選択した．
  7. Mouseover Dictionaryを使う
    「Alt*m」でサイドバーにMouseover Dictionaryが出るので，後は読みたい英文にマウスオーバするだけ!

PDF で Mouseover Dictinonary を使う¶

  1. firefoxのアドレスバーに「about:config」を入力
  2. 細心の注意を払って使用するをクリックする．
  3. pdfjs.disabledをfalseにする．
  4. Firefoxの環境設定->アプリケーション
  5. PDF文書を「Firefoxでプレビュー表示」にする．
