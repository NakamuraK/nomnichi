Githubの使い方

乃村研では，Githubでリポジトリを管理している．
ここでは，Githubを使うための準備と，使い方について述べる．

アカウント登録

  1. Github(https://github.com/)にアカウントを作成する．
  2. 作成したアカウントをnomlabのTeamsに追加してもらう．
    (追加はnomlabのownerグループに所属する人(乃村先生とM2)のみ行える．)
  3. 自身のアカウントに，Githubを使う環境の公開鍵を登録する．

公開鍵の登録

  1. 自身の環境で，以下のコマンドを実行し，公開鍵を作成する
         $ ssh-keygen -t rsa -f .ssh/id_rsa -N パスフレーズ
  2. 公開鍵は，~/.ssh/id_rsa.pub というファイルに保存される．
    このファイルの中身をコピーする．(改行などに注意)
  3. GitHubにログインし， AccountSettings >> SSH-Keys >> Add SSH key
    に移動し，Keyを貼り付ける．Titleは環境が分かるように自分で適当に決める．

これで準備は完了である．
次に，Githubの利用方法を述べる．
例として，ノムニチ(Yesod)を用いて説明する．

Githubの使い方

  1. 任意のディレクトリで，以下のコマンドを実行する．
       $ git clone git@github.com:nomlab/nomnichi.git
  2. 作業が終わったら，addしてcommitしてpushすれば良い．
