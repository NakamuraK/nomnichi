はじめに

Subversionとは，日々更新されるファイルやディレクトリを管理するバージョン管理システムである．
ファイルのバージョンに関する情報は，リポジトリと呼ばれるディレクトリに置かれる．
リポジトリにはファイルやディレクトリに関する変更内容が記録される．
このため，古いバージョンのファイルのデータを復活させたり，変更履歴を調べたりすることが出来る．
このページではSubversionの使用方法の例を示す．

Subversionの使用例

以下に例として，乃村研HPに個人ページを追加する方法を示す．
(1) 作業コピーの作成
  $ svn checkout svn+ssh://tsubame.swlab.cs.okayama-u.ac.jp/var/svn/lastnote/experimentals/nomlab/nomnichi

(2) 移動
  $ cd nomnichi/public_html/users

(3) 自分のアカウント名のディレクトリの作成
  $ mkdir アカウント名

(4) (3)のディレクトリの中にindex.htmlを作成(index.htmlは先輩のものを参考に作成)
(注) 文字のエンコーディングをUTF-8にする．

(5) レポジトリに作成したディレクトリを追加する準備
  $ cd ..
  $ svn add 追加したディレクトリ

(6) 作業コピーの変更点をレポジトリに送る
  $ svn commit -m "変更点に関するコメント"

追加完了．
