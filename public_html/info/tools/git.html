はじめに

このページでは，tsubame上のホームディレクトリでnomnichiの開発を行う方法を例に，Gitを利用した開発方法について説明する．

Gitを利用したnomnichiの開発

(1) ローカルリポジトリを作成 
    git clone ssh://tsubame.swlab.cs.okayama-u.ac.jp/home/home_directory_name/nomnichi /home/home_directory_name/
(2) ファイルを編集
(3) 編集内容を確認
  • ローカルリポジトリの状態を確認
    • git status
  • ファイルの変更点を確認
    • git diff
  • Web上で確認
    • 作成したWebページをWeb上で確認を参照
(4) 編集したファイルをコミット
  • 編集したファイルをindexにステージ
    • git add file_path
  • indexにある変更点をコミット
    • git commit
  • エディタでコミットメッセージを入力し，保存して修了
(5) コミットを確認
  • コミットログを確認
    • git log
  • 編集内容を確認
    • git show

(6) リモートブランチに変更点を送信
  • リモートのmasterリポジトリに変更点を送信
    • git push origin master
