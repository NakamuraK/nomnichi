PDFに図を入れるカッコイイ方法

作成日：2013年2月1日
最終更新日：2013年2月1日
村田　裕哉

はじめに

このページでは，PDFに図を入れるカッコイイ方法を紹介する．
これにより，epsファイルをいくつも作る必要がなくなる．

画像ファイルの準備

  1. 画像ファイルをPDFで保存する．複数の画像を1つのPDFとして良い．

Makefileの入手

  1. 乃村先生直伝のMakefileを入手する．
  2. 取得したMakefileをTexファイルがあるディレクトリに置く．
  3. Makefileを各自の環境に合わせて編集する．

Makefileの編集

  1. Texファイルの文字コードに合わせる．
    文字コードがsjisの場合は，以下のように変更する．
    2. PLATEX = platex-sjis -interaction=nonstopmode
  2. PROJECTにTexファイルの名前を指定する．thesis.texの場合は以下のように書く． 
    3. PROJECT = thesis
  3. 画像を置くディレクトリとファイル名をFIG_MASTER_FILEに指定する．
    4. FIG_MASTER_FILE = fig/fig.pdf
  4. PDFの何ページ目から画像を取り出すかを指定する．
    例えば，fig.pdfの1～5ページから画像を取り出す場合は，以下のように書く．
    これにより，各画像は fig1.pdf，fig2.pdf…というファイル名になる．
    FIG_PAGES = 1 2 3 4 5

Texファイルの編集

サンプルTexファイルを参考にすると，よりカッコイイ．
  1. usepackageを変更する．
    2. \usepackage[dvipdfm]{graphics}
  2. 画像の参照方法を変更する．この時，clipを忘れないようにする．
    3. \includegraphics[clip,widht=1\columnwidth]{.fig/fig1.pdf}

実際に使う

makeを叩くだけで，画像が個々のpdfに分割されたりあれされたりしてtexファイルがpdfになって完成する． 
$ make
